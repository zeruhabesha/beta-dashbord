🚀 Quick Start (3 Steps)

  1. Configure OpenSearch Dashboards
    cp opensearch-dashboards-config/opensearch_dashboards.yml /path/to/config/
    systemctl restart opensearch-dashboards
  2. Ingest Sample Data
    pip install opensearch-py
    python opensearch-dashboards-config/ingest-sample-data.py
    This will create 15,000 sample security events across 3 tenants
  3. Import Dashboards

    Open http://localhost:5601

    Navigate to: Stack Management → Saved Objects → Import

    Select: saved-objects-export.ndjson
